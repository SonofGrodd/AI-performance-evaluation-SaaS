# Fredan Frontend – React Client App

This is the modern React frontend for the Fredan performance platform. It supports:

- Auth & protected routes
- Tailwind dashboard layout
- Employee & Manager views
- API integration with Express backend
- Responsive UI & state management

---

## 🛠️ Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Axios
- Context API

---
